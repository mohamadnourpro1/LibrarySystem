# Library System (HTML/CSS/JS)

## فكرة المشروع
نظام مكتبة بسيط لإدارة الكتب، الأعضاء، والإعارات باستخدام واجهات فقط (بدون Backend) مع تخزين البيانات في localStorage.

## أعضاء الفريق والأدوار
- محمد نور المحمد — Team Lead / Integration / Dashboard
- محمد مرهف العكلة — Books Module (CRUD + Search)
- احمد امين حبار — Members + Loans Module (CRUD + Lending/Return)

## التقنيات المستخدمة
- HTML
- CSS
- JavaScript (Vanilla)
- localStorage
- Git / GitHub
- Jira

## طريقة تشغيل المشروع
1) افتح مجلد `src`
2) شغل `index.html` مباشرة على المتصفح  
> يفضّل تشغيله عبر Live Server (VS Code) لتجربة أفضل.

## روابط الصفحات
- Dashboard: `src/pages/dashboard.html`
- Books: `src/pages/books.html`
- Members: `src/pages/members.html`
- Loans: `src/pages/loans.html`
